# 致谢名单

这里是「大神之路」特别感谢区，专门用来感谢那些让我这个博客能跑起来、还能变好看的项目和作者们。

## 主题与框架

- **vitepress-theme-curve** 主题

  - 作者：[@imsyy](https://github.com/imsyy)
  - 仓库：<https://github.com/imsyy/vitepress-theme-curve>
  - 说明：本博客的整体界面和交互，都基于这个优秀的 VitePress 主题进行二次改造。如果没有这个主题，我大概率还在纠结「要不要先自己写一个主题」。

- **VitePress**
  - 仓库：<https://github.com/vuejs/vitepress>
  - 说明：整站的文档/博客系统底层框架，轻量、好用、写起来很舒服。

## 技术与生态

- **Vue / Vue3 / React / Node.js / NestJS / MySQL / Docker**
  - 这些是「大神之路」主要围绕的技术栈，也是我现在正在持续学习和踩坑的方向。
  - 文章里提到的很多思路、实践，都离不开社区中无数前辈和文章的启发。

- 如果你是我引用到的项目作者、文章作者，或者你发现自己被我「悄悄学习」了，也感谢你愿意把知识分享出来，让像我这样的后来者能少走很多弯路。
