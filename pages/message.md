# 畅所欲言

这里是「大神之路」的留言角落。

你可以在这里：

- 随便聊聊最近在学什么、在踩什么坑；
- 吐槽一下写代码时遇到的奇怪 bug；
- 给我提一些想看的文章方向（比如 Vue3 / React / NestJS / MySQL / Docker 等）；
- 或者单纯打个招呼，证明不是只有我自己在刷新这个站。

目前评论功能取决于站点的实际配置：

- 如果你在下方看到了评论框，就可以直接留言；
- 如果暂时没有评论入口，那就先当作一个「预备留言页」，后面等我折腾好评论系统再开放。

不管怎样，很高兴你愿意在这里停留片刻。
